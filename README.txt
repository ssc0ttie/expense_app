## Expense Entry Form

This project consists of a user form developed using Google Scripts. The form is designed to collect expense entries and is linked to a Google Sheets file for efficient data storage and management.

### Features:

- **Expense Entry:** Users can easily input various expense details such as date, item description, amount, and category through the form.
- **Integration with Google Sheets:** The form seamlessly integrates with Google Sheets, allowing the collected data to be automatically populated into a specified spreadsheet for further analysis and tracking.
- **User-Friendly Interface:** The form is designed with simplicity and ease of use in mind, ensuring a smooth and intuitive experience for users.

### Usage:

To use the expense entry form:
1. Access the provided Google Forms link.
2. Fill out the required fields with the relevant expense details.
3. Submit the form to automatically record the entry in the connected Google Sheets file.

### Technologies Used:

- **Google Apps Script:** Used to create and deploy the user form and handle the submission of data.
- **Google Sheets:** Integrated with the form to store and organize the collected expense entries.

### Getting Started:

To set up the expense entry form:
1. Clone or download the project repository.
2. Follow the instructions in the `setup.md` file to deploy the form and configure the integration with Google Sheets.
3. Start collecting expense entries efficiently and effectively!
